## 快速开始
- 安装MySQL数据库
- 修改root密码为123456
- 创建数据库:bbs
- 在bbs数据库下执行脚本schema.sql
- 安装git版本控制工具
- 安装jdk1.7 
- 安装IntelliJ IDEA
- 打开IDEA, 从 https://github.com/kangyonggan/bbs 把项目拉取到本地
- mvn clean install 打包项目
- mvn tomcat7:run 启动项目, 也可以把war包扔到tomcat/webapps目录下运行
